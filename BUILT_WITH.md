# Built with Storm

Storm TUI powers real-world applications:

## Official Examples

| App | Description | Run |
|-----|-------------|-----|
| **Storm Code** | AI coding agent with streaming, diffs, and approval prompts | `npx tsx examples/storm-code/index.tsx` |
| **Storm Ops** | Operations monitoring dashboard with live metrics | `npx tsx examples/storm-ops/index.tsx` |
| **Letta Clone** | AI agent with persistent memory and tool use | `npx tsx examples/letta-clone/index.tsx` |
| **Launch Demo** | Cinematic auto-playing agent scenario | `npx tsx examples/demo-launch.tsx` |

## Showcase Templates

| Template | Style | Run |
